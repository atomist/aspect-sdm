Const K8sSpecs

K8sSpecs: object
  • Defined in lib/aspect/k8s/specAspect.ts:158

Aspect emitting 0 or more Kubernetes resource spec fingerprints. Each fingerprint represents a single resource spec. Across multiple repositories, the fingerprints can be compared to check for drift in resources that provide core functionality like external-dns and nginx-ingress. If a repo does not contain any JSON or YAML files in the root of the project that look like Kubernetes resource specs, no fingerprints are emitted.
